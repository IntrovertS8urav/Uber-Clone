# Uber Clone (Full Stack)

A full-stack Uber clone built with the MERN stack, featuring real-time socket communication and map integration.

## Project Structure
- **/Backend**: Node.js & Express server with MongoDB, JWT Auth, and Socket.io.
- **/Frontend**: React.js / Vite application for the user interface.

## Features
- User & Captain Authentication
- Real-time ride requests using Sockets
- Map & Location services integration
- Ride management logic

## How to Run Locally

### Backend:
1. `cd Backend`
2. `npm install`
3. Create a `.env` file with your `MONGO_URL` and `JWT_SECRET`.
4. `npm start`

### Frontend:
1. `cd Frontend`
2. `npm install`
3. `npm run dev`
